# How to setup the project

1. Clone the project to your local machine
	```
	git clone
	```
2. Navigate to the project's folder
	```
	cd netium-workshop	
	```
3. Install the dependencies
	```
	npm install	
	```
	or
	```
	yarn install	
	```
4. Start the development server
	```
	next dev	
	```
	or
	```
	yarn dev	
	```
